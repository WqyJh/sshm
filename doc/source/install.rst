Installation
============

sshx is on PyPI, and can be installed with pip::

    pip install sshx


Supported platforms
-------------------
- Linux
- macOS
- WSL/cygwin/msys2 on Windows

**Attention:**

- Windows CMD/PowerShell support was removed.
- Python 2 support was removed.


Requirements
------------
- Python >= 3.6
- openssh-clients: ``ssh``, ``scp``, ``ssh-keygen``
